🔑 Connetti GitHub

Per salvare i progetti direttamente su GitHub, hai bisogno di un Personal Access Token.

1. Vai su github.com/settings/tokens/new
2. Scegli "Tokens (classic)"
3. Spunta il permesso repo
4. Clicca Generate token e incollalo qui sotto.
